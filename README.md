# Vacation Pay Calculator
## Описание
Микросервисное приложение на Spring Boot для расчета отпускных. Приложение рассчитывает сумму отпускных на основе средней заработной платы сотрудника.
## Функциональность
### Базовый расчет: 
Принимает среднюю зарплату и количество дней отпуска.
### Расчет по датам (Доп. задание): 
Принимает точные даты ухода в отпуск.
Учитываются официальные праздничные дни (нерабочие).
Выходные дни оплачиваются в обычном порядке.
- Валидация: Проверка входных данных (отрицательные суммы, пустые поля, логические ошибки дат).
- Структура: Чистая архитектура с использованием паттерна "Стратегия" (Strategy Pattern).

## Установка и запуск
Требования
- Установленный JDK 11.

Выполните команду:

- mvn clean install && mvn spring-boot:run
- Приложение запустится на порту 8081.

## Использование API
### Единый эндпоинт:

- GET /calculate

### Параметры запроса
| Параметр        | Тип         | Обязательный | Описание                                 |
|-----------------|-------------|--------------|------------------------------------------|
| `averageSalary` | `Integer`   | Да           | Средняя заработная плата                 |
| `vacationDays`  | `Integer`   | Да           | Количество дней отпуска                  |
| `vacationDate`  | `LocalDate` | Нет*         | Дата начала отпуска (формат: dd-MM-yyyy) |
 \* Не обязательно в случае расчета без учета выходных дней

### Примеры запросов
1. Простой расчет (по дням):

GET http://localhost:8081/calculate?averageSalary=85000&vacationDays=16

#### Ответ:

{
<br>"calculationResult": 46416.32
<br>}

2. Расчет по датам (с учетом праздников):
   Пример: Отпуск на майские праздники (1-3 мая 2026). 1 мая не оплачивается.

GET http://localhost:8081/calculate?averageSalary=85000&vacationDays=3&vacationDate=01.05.2026

#### Ответ (будут оплачены только 2 и 3 мая):

{<br>"calculationResult": 5802.04<br>}

3. Ошибка валидации:
   
GET http://localhost:8081/calculate?averageSalary=-500&vacationDays=10

#### Ответ:

{<br>"averageSalary": "Средняя зарплата должна быть больше нуля"<br>}
